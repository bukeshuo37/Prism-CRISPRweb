# Prism-CRISPRweb

[![DOI](https://img.shields.io/badge/DOI-10.5281%2Fzenodo.20278480-blue)](https://doi.org/10.5281/zenodo.20278480)

Prism-CRISPR Web is a web-based platform centered on human single-base mutations.  
It designs candidate sgRNAs for mutation information and predicts/ranks editing outcomes using the selected editor.

## Online Demo

👉 https://prism-crispr-web.vercel.app 

👉 https://www.crispr-web.com


## Usage

1. Open the web server using either link above. 
2. Enter disease-related mutation information in **Mutation Information**, or input an **OMIM_ID**.
3. Select the desired **editor**.
4. Choose the ranking criterion.
5. Click **Search**.
6. View the ranked sgRNA design and prediction results.
